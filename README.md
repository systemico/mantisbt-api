MantisBT PHP Client
===================

This is a php client for MantisBT / MantisHub.  It's goal is to simplify interacting
with SOAP API from PHP.

Requirements
------------
- MantisHub (supported)
- MantisBT 2.x.y (supported)
- MantisBT 1.3.x (supported)
- MantisBT 1.2.16+ (use latest 1.2.x release)
